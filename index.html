Detailed instructions
Follow easy step guide through to setup your pixelbot

1. Download vmware player 7.1.0. (try not to download v12 or latest version because it's buggy and unstable) then install it.
DOWNLOAD LINK: http://filehippo.com/download_vmware_player/60123/



2. Download windows 7 lite located inside folder (using different windows7.iso build the bot may wont run. make sure you use my provided windows7.iso) to download it you need bittorent or utorrent client
DOWNLOAD LINK: http://www.bittorrent.com/






3.After downloading windows7. Now open Vmware player create new virtual machine and select windows7ultimate.iso






4.Just click next, next, it will ask you product key. just skip it because our windows 7 is already pre-activated, and then set vmware disk drive to 20gb (we dont need to waste huge large disk)






5. click next, set processor to 2core and 2gb memory (this is enough to run windows7) then on windows 7 installation MAKE SURE YOU CHOOSE WINDOWS 7 64-BIT (32bit wont work)







6. Now after installing Windows 7, VmwareTools will automatically install then will restart.


7. go to your host ragnarok main folder and drag and drop your or copy paste whole ragnarok to copy inside vmware windows7


8.also drag and drop copy lenzki bot folder inside vmware.

But First you need to edit data.grf to modify monster sprites

9. go to your main ragnarok online folder and create copy of data.grf (so you can revert back the original sprites)





10. open folder lenzki bot\grf builder\ and run grfbuilder.exe




11. on grfbuilder,click file -> open -> locate your ragnarok online folder and select data.grf




12. wait to finish loading, then finally click 'MERGE GRF', go back and locate lenzki Bot folder and select monster.grf, after merging you can now close grfbuilder





13. run your ragnarok online and login ingame go to prontera south field and check if the poring monster changed to green pixel




14. you can now finally run ROBOT.EXE press start. enjoy!







QUICK HOTKEYS

Press END = Stop
Press INSERT = Pause
