Setup instructions
Follow easy step guide through to setup your pixelbot

First you need to edit data.grf to modify monster sprites

1. go to your main ragnarok online folder and create copy of data.grf (so you can revert back the original sprites)





2. open folder lenzki bot folder and install grf editor.exe




3. after installing, run GRF EDITOR as administrator, click file -> open -> locate your ragnarok online folder and select data.grf





4. click Edit -> Merge -> Browse -> Load. locate lenzki bot folder then select MONSTER.GRF, and finally click Merge button and wait to finish loading.






5. go to your ragexe.exe and ragnarok.exe set compatibility to windows xp sp3 then click ok







open setup.exe and follow these settings, make sure you unchecked mouse exclusive or your mouseclicks will not work

6. run your ragnarok online and login ingame go to prontera south field or novice training ground, check if the poring monster changed to green pixel (note trial version only change poring monster)




run task killer.exe and follow the video step watch on 720p HD FULL SCREEN FOR CLEARER INSTRUCTION



7. IMPORTANT RIGHT CLICK PIXEL RO BOT.EXE AND RUN AS ADMIN press start button and. enjoy!
